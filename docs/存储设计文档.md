# MiniMeeting 存储设计文档

## 1. 存储概述

MiniMeeting 是一个轻量级实时音视频通信系统，当前版本采用内存存储方案，所有数据在服务器重启后会丢失。后续版本可根据需要升级为持久化存储方案。

---

## 2. 服务器端数据结构

### 2.1 在线用户数据

存储当前连接的所有在线用户信息。

```javascript
// 数据结构
const onlineUsers = new Map();

// 结构定义
onlineUsers = {
    userId => {
        ws: WebSocket,           // WebSocket 连接对象
        userName: string,        // 用户显示名称
        roomId: string | null,   // 当前所在房间ID，null 表示未加入房间
        loginTime: number,       // 登录时间戳（毫秒）
        lastHeartbeat: number    // 最后心跳时间戳（毫秒）
    }
}
```

**示例：**
```javascript
onlineUsers = Map(3) {
    'user_001' => {
        ws: WebSocket { ... },
        userName: '张三',
        roomId: 'room_001',
        loginTime: 1700000000000,
        lastHeartbeat: 1700000030000
    },
    'user_002' => {
        ws: WebSocket { ... },
        userName: '李四',
        roomId: 'room_001',
        loginTime: 1700000010000,
        lastHeartbeat: 1700000031000
    },
    'user_003' => {
        ws: WebSocket { ... },
        userName: '王五',
        roomId: null,
        loginTime: 1700000020000,
        lastHeartbeat: 1700000032000
    }
}
```

### 2.2 房间数据

存储所有活跃的会议室信息。

```javascript
// 数据结构
const rooms = new Map();

// 结构定义
rooms = {
    roomId => {
        roomId: string,          // 房间唯一标识
        roomName: string,        // 房间名称
        creatorId: string,       // 创建者用户ID
        users: Set<string>,      // 房间内用户ID集合
        createdAt: number,       // 创建时间戳（毫秒）
        maxUsers: number         // 最大人数限制，默认 4
    }
}
```

**示例：**
```javascript
rooms = Map(2) {
    'room_001' => {
        roomId: 'room_001',
        roomName: '项目讨论会',
        creatorId: 'user_001',
        users: Set(2) { 'user_001', 'user_002' },
        createdAt: 1700000005000,
        maxUsers: 4
    },
    'room_002' => {
        roomId: 'room_002',
        roomName: '周会',
        creatorId: 'user_003',
        users: Set(1) { 'user_003' },
        createdAt: 1700000015000,
        maxUsers: 4
    }
}
```

### 2.3 心跳计时器

存储用户心跳检测计时器。

```javascript
// 数据结构
```

### 2.4 用户ID与WebSocket映射

用于通过 WebSocket 连接快速查找用户。


## 4. 数据操作接口

### 4.1 用户管理接口

```javascript
class UserManager {
    constructor() {
        this.users = new Map();
        this.wsToUser = new Map();
    }

    /**
     * 添加用户
     * @param {string} userId - 用户ID
     * @param {WebSocket} ws - WebSocket连接
     * @param {string} userName - 用户名称
     */
    addUser(userId, ws, userName) {
        this.users.set(userId, {
            ws,
            userName,
            roomId: null,
            loginTime: Date.now(),
            lastHeartbeat: Date.now()
        });
        this.wsToUser.set(ws, userId);
    }

    /**
     * 移除用户
     * @param {string} userId - 用户ID
     */
    removeUser(userId) {
        const user = this.users.get(userId);
        if (user) {
            this.wsToUser.delete(user.ws);
            this.users.delete(userId);
        }
    }

    /**
     * 获取用户信息
     * @param {string} userId - 用户ID
     * @returns {Object|null} 用户信息对象
     */
    getUser(userId) {
        return this.users.get(userId) || null;
    }

    /**
     * 通过 WebSocket 获取用户ID
     * @param {WebSocket} ws - WebSocket连接
     * @returns {string|null} 用户ID
     */
    getUserIdByWs(ws) {
        return this.wsToUser.get(ws) || null;
    }

    /**
     * 更新用户所在房间
     * @param {string} userId - 用户ID
     * @param {string|null} roomId - 房间ID
     */
    setUserRoom(userId, roomId) {
        const user = this.users.get(userId);
        if (user) {
            user.roomId = roomId;
        }
    }

    /**
     * 更新心跳时间
     * @param {string} userId - 用户ID
     */
    updateHeartbeat(userId) {
        const user = this.users.get(userId);
        if (user) {
            user.lastHeartbeat = Date.now();
        }
    }

    /**
     * 获取在线用户数量
     * @returns {number} 在线用户数
     */
    getOnlineCount() {
        return this.users.size;
    }

    /**
     * 获取所有在线用户ID
     * @returns {string[]} 用户ID数组
     */
    getAllUserIds() {
        return Array.from(this.users.keys());
    }

    /**
     * 检查用户是否在线
     * @param {string} userId - 用户ID
     * @returns {boolean}
     */
    isOnline(userId) {
        return this.users.has(userId);
    }
}
```

### 4.2 房间管理接口

```javascript
class RoomManager {
    constructor() {
        this.rooms = new Map();
    }

    /**
     * 生成房间ID
     * @returns {string} 房间ID
     */
    generateRoomId() {
        return 'room_' + Date.now().toString(36) + '_' + 
               Math.random().toString(36).substr(2, 9);
    }

    /**
     * 创建房间
     * @param {string} roomName - 房间名称
     * @param {string} creatorId - 创建者ID
     * @returns {Object} 创建的房间对象
     */
    createRoom(roomName, creatorId) {
        const roomId = this.generateRoomId();
        const room = {
            roomId,
            roomName: roomName || '会议室',
            creatorId,
            users: new Set([creatorId]),
            createdAt: Date.now(),
            maxUsers: 4
        };
        this.rooms.set(roomId, room);
        return room;
    }

    /**
     * 获取房间信息
     * @param {string} roomId - 房间ID
     * @returns {Object|null} 房间对象
     */
    getRoom(roomId) {
        return this.rooms.get(roomId) || null;
    }

    /**
     * 加入房间
     * @param {string} roomId - 房间ID
     * @param {string} userId - 用户ID
     * @returns {boolean} 是否成功
     */
    joinRoom(roomId, userId) {
        const room = this.rooms.get(roomId);
        if (!room) return false;
        if (room.users.size >= room.maxUsers) return false;
        room.users.add(userId);
        return true;
    }

    /**
     * 离开房间
     * @param {string} roomId - 房间ID
     * @param {string} userId - 用户ID
     * @returns {boolean} 是否成功
     */
    leaveRoom(roomId, userId) {
        const room = this.rooms.get(roomId);
        if (!room) return false;
        room.users.delete(userId);
        
        // 如果房间为空，删除房间
        if (room.users.size === 0) {
            this.rooms.delete(roomId);
        }
        return true;
    }

    /**
     * 获取房间内所有用户
     * @param {string} roomId - 房间ID
     * @returns {string[]} 用户ID数组
     */
    getRoomUsers(roomId) {
        const room = this.rooms.get(roomId);
        return room ? Array.from(room.users) : [];
    }

    /**
     * 获取房间当前人数
     * @param {string} roomId - 房间ID
     * @returns {number} 用户数量
     */
    getRoomUserCount(roomId) {
        const room = this.rooms.get(roomId);
        return room ? room.users.size : 0;
    }

    /**
     * 检查房间是否存在
     * @param {string} roomId - 房间ID
     * @returns {boolean}
     */
    roomExists(roomId) {
        return this.rooms.has(roomId);
    }

    /**
     * 检查房间是否已满
     * @param {string} roomId - 房间ID
     * @returns {boolean}
     */
    isRoomFull(roomId) {
        const room = this.rooms.get(roomId);
        return room ? room.users.size >= room.maxUsers : true;
    }

    /**
     * 获取所有房间列表
     * @returns {Array} 房间信息数组
     */
    getAllRooms() {
        return Array.from(this.rooms.values()).map(room => ({
            roomId: room.roomId,
            roomName: room.roomName,
            userCount: room.users.size,
            creatorId: room.creatorId,
            createdAt: room.createdAt
        }));
    }

    /**
     * 删除房间
     * @param {string} roomId - 房间ID
     */
    deleteRoom(roomId) {
        this.rooms.delete(roomId);
    }

    /**
     * 获取房间总数
     * @returns {number}
     */
    getRoomCount() {
        return this.rooms.size;
    }
}
```

## 6. 数据清理策略

### 6.1 心跳检测清理

```javascript
class HeartbeatManager {
    constructor(userManager, roomManager) {
        this.userManager = userManager;
        this.roomManager = roomManager;
        this.checkInterval = 30000;  // 检查间隔：30秒
        this.timeout = 90000;        // 超时时间：90秒
    }

    /**
     * 启动心跳检测
     */
    start() {
        setInterval(() => this.checkTimeout(), this.checkInterval);
    }

    /**
     * 检查超时用户
     */
    checkTimeout() {
        const now = Date.now();
        const timeoutUsers = [];

        this.userManager.users.forEach((user, userId) => {
            if (now - user.lastHeartbeat > this.timeout) {
                timeoutUsers.push(userId);
            }
        });

        // 清理超时用户
        timeoutUsers.forEach(userId => {
            this.cleanupUser(userId);
        });
    }

    /**
     * 清理用户资源
     * @param {string} userId - 用户ID
     */
    cleanupUser(userId) {
        const user = this.userManager.getUser(userId);
        if (!user) return;

        // 如果用户在房间中，需要离开房间
        if (user.roomId) {
            const roomUsers = this.roomManager.getRoomUsers(user.roomId);
            
            // 通知房间内其他用户
            roomUsers.forEach(otherUserId => {
                if (otherUserId !== userId) {
                    const otherUser = this.userManager.getUser(otherUserId);
                    if (otherUser && otherUser.ws) {
                        otherUser.ws.send(JSON.stringify({
                            type: 'userLeft',
                            roomId: user.roomId,
                            userId: userId
                        }));
                    }
                }
            });

            // 用户离开房间
            this.roomManager.leaveRoom(user.roomId, userId);
        }

        // 移除用户
        this.userManager.removeUser(userId);
    }
}
```

### 6.2 房间自动清理

- 当房间内所有用户离开时，自动删除房间
- 不需要额外的定时清理任务

## 7. 持久化扩展方案（可选）

如需持久化存储，可采用以下方案：

### 7.1 Redis 方案

```javascript
// 用户数据存储
// Key: user:{userId}
// Value: { userName, roomId, loginTime, lastHeartbeat }

// 房间数据存储
// Key: room:{roomId}
// Value: { roomName, creatorId, createdAt, maxUsers }

// 房间用户集合
// Key: room:{roomId}:users
// Value: Set { userId1, userId2, ... }
```

### 7.2 数据库方案（SQLite/MySQL）

```sql
-- 用户表
CREATE TABLE users (
    user_id VARCHAR(64) PRIMARY KEY,
    user_name VARCHAR(32),
    room_id VARCHAR(64),
    login_time BIGINT,
    last_heartbeat BIGINT
);

-- 房间表
CREATE TABLE rooms (
    room_id VARCHAR(64) PRIMARY KEY,
    room_name VARCHAR(64),
    creator_id VARCHAR(64),
    created_at BIGINT,
    max_users INT DEFAULT 4
);

-- 房间用户关联表
CREATE TABLE room_users (
    room_id VARCHAR(64),
    user_id VARCHAR(64),
    joined_at BIGINT,
    PRIMARY KEY (room_id, user_id)
);
```

## 8. 存储容量估算

### 8.1 单用户内存占用

| 字段 | 类型 | 大小（估算） |
|------|------|--------------|
| userId | string | 32 B |
| userName | string | 32 B |
| roomId | string | 32 B |
| loginTime | number | 8 B |
| lastHeartbeat | number | 8 B |
| ws | object | ~100 B |
| **总计** | - | **~212 B** |

### 8.2 单房间内存占用

| 字段 | 类型 | 大小（估算） |
|------|------|--------------|
| roomId | string | 32 B |
| roomName | string | 64 B |
| creatorId | string | 32 B |
| users | Set | 32 B × 4 = 128 B |
| createdAt | number | 8 B |
| maxUsers | number | 4 B |
| **总计** | - | **~268 B** |

### 8.3 总内存估算

假设同时在线 100 用户，25 个房间：

- 用户数据：100 × 212 B ≈ 21 KB
- 房间数据：25 × 268 B ≈ 7 KB
- 总计：约 28 KB

内存占用极小，无需担心性能问题。
