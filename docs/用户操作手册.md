# MiniMeeting 用户操作手册

## 1. 产品简介

MiniMeeting 是一款轻量级实时音视频通信软件，支持多人语音/视频通话，具有低延迟、弱网适应能力强等特点。

### 1.1 主要功能

- 实时语音通话
- 实时视频通话
- 多人会议室（最多 4 人）
- 网络质量监控
- 设备管理

---

## 2. 系统要求

### 2.1 硬件要求

| 组件 | 最低配置 | 推荐配置 |
|------|----------|----------|
| CPU | 双核 2.0GHz | 四核 2.5GHz+ |
| 内存 | 4 GB | 8 GB+ |
| 摄像头 | 720p | 1080p |
| 麦克风 | 内置麦克风 | 外置麦克风 |
| 网络 | 1 Mbps 上/下行 | 5 Mbps+ 上/下行 |

### 2.2 软件要求

- 操作系统：Windows 10/11 64位
- 音频驱动：正常工作的声卡驱动
- 网络环境：支持 UDP 协议

---

## 3. 安装指南

### 3.1 下载安装

1. 从官方网站下载最新安装包
2. 双击安装包运行安装程序
3. 选择安装目录，点击"下一步"
4. 等待安装完成
5. 点击"完成"启动应用

### 3.2 首次启动

首次启动时，系统可能会弹出以下权限请求：

- **麦克风权限**：请允许，否则无法进行语音通话
- **摄像头权限**：请允许，否则无法进行视频通话
- **防火墙权限**：请允许，否则可能无法建立连接

---

## 4. 快速入门

### 4.1 登录

1. 启动 MiniMeeting 应用
2. 在登录界面输入您的用户名
3. 点击"登录"按钮

![登录界面](images/login.png)

### 4.2 创建会议室

1. 登录后进入主界面
2. 点击"创建房间"按钮
3. 输入房间名称（可选）
4. 点击"确认"创建房间
5. 系统将自动分配房间号

![创建房间](images/create-room.png)

### 4.3 加入会议室

**方式一：从房间列表加入**
1. 在房间列表中找到目标房间
2. 点击"加入"按钮
3. 等待连接建立

**方式二：通过房间号加入**
1. 点击"加入房间"按钮
2. 输入房间号
3. 点击"确认"加入

![加入房间](images/join-room.png)

### 4.4 进行通话

进入房间后，系统会自动建立音视频连接：

- **语音通话**：自动开启，您可以直接说话
- **视频通话**：默认开启摄像头，可在控制栏关闭

### 5.2 控制栏功能

| 按钮 | 功能 | 说明 |
|------|------|------|
| 🎤 | 麦克风开关 | 开启/关闭麦克风，关闭后对方听不到您的声音 |
| 📷 | 摄像头开关 | 开启/关闭摄像头，关闭后对方看不到您的画面 |
| 🔊 | 扬声器开关 | 开启/关闭扬声器，关闭后您听不到对方声音 |
| 📊 | 网络状态 | 显示/隐藏网络质量详情面板 |
| 📞 | 挂断 | 离开房间，结束通话 |

### 5.3 网络状态面板

点击网络状态按钮可查看详细网络信息：

| 指标 | 说明 | 正常范围 |
|------|------|----------|
| RTT | 往返延迟 | < 150ms（良好）<br>150-300ms（一般）<br>> 300ms（较差） |
| 丢包率 | 数据包丢失比例 | < 1%（良好）<br>1-5%（一般）<br>> 5%（较差） |
| 发送码率 | 上行带宽使用 | 根据视频质量变化 |
| 接收码率 | 下行带宽使用 | 根据视频质量变化 |
| 连接类型 | ICE 连接方式 | host（直连）> srflx（反射）> relay（中继） |

## 6. 设置说明

### 6.1 音频设置

进入"设置 > 音频"可配置：

- **麦克风**：选择输入设备，调节音量
- **扬声器**：选择输出设备，调节音量
- **降噪**：开启/关闭环境降噪
- **回声消除**：开启/关闭回声消除（建议开启）

### 6.2 视频设置

进入"设置 > 视频"可配置：

- **摄像头**：选择视频输入设备
- **分辨率**：选择视频分辨率
  - 高清（640x480）
  - 标清（320x240）
- **帧率**：选择视频帧率
  - 流畅（30fps）
  - 省流（15fps）

### 6.3 网络设置

进入"设置 > 网络"可配置：

- **信令服务器**：设置信令服务器地址
- **STUN 服务器**：NAT 穿透服务器
- **TURN 服务器**：中继服务器（可选）

## 7. 多人通话指南

### 7.1 多人房间特点

MiniMeeting 采用 Mesh 拓扑结构，多人通话时：

- 每人需要与其他所有参会者建立连接
- 建议最多 4 人同时通话
- 每增加一人，带宽需求相应增加

### 7.2 多人通话建议

| 参会人数 | 建议带宽 | 视频设置 |
|----------|----------|----------|
| 2 人 | 2 Mbps | 640x480, 30fps |
| 3 人 | 4 Mbps | 640x480, 30fps |
| 4 人 | 6 Mbps | 320x240, 15fps |

## 8. 常见问题

### 8.1 无法登录

**问题现象**：点击登录后无响应或提示连接失败

**解决方案**：
1. 检查网络连接是否正常
2. 检查信令服务器地址是否正确
3. 检查防火墙是否阻止了应用
4. 尝试更换网络环境

### 8.2 听不到对方声音

**问题现象**：能看视频但听不到声音

**解决方案**：
1. 检查扬声器是否被关闭（控制栏按钮）
2. 检查系统音量是否静音
3. 检查音频输出设备是否正确
4. 请对方检查麦克风是否开启

### 8.3 对方听不到我的声音

**问题现象**：对方反馈听不到我的声音

**解决方案**：
1. 检查麦克风是否被关闭（控制栏按钮）
2. 检查麦克风音量是否过低
3. 检查音频输入设备是否正确
4. 测试麦克风是否正常工作

### 8.4 视频画面卡顿

**问题现象**：视频卡顿或延迟严重

**解决方案**：
1. 检查网络质量面板的延迟和丢包率
2. 降低视频分辨率和帧率
3. 关闭不必要的网络应用
4. 尝试关闭摄像头，仅使用语音

### 8.5 连接一直显示"连接中"

**问题现象**：长时间无法建立连接

**解决方案**：
1. 检查 STUN/TURN 服务器配置
2. 检查网络是否支持 UDP 协议
3. 尝试切换网络（如从 WiFi 切换到有线）
4. 联系管理员检查服务器状态

### 8.6 多人通话时崩溃

**问题现象**：多人通话时程序崩溃或卡死

**解决方案**：
1. 确保系统满足最低硬件要求
2. 降低视频分辨率
3. 关闭其他占用资源的应用
4. 检查内存是否充足
